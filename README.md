# Ext Métodos Numéricos – Regra do Trapézio

## Descrição
Programa em Python para calcular integrais usando o **método do trapézio**, incluindo a estimativa de **erro de truncamento** e **erro de arredondamento**. Ideal para estudos em **Métodos Numéricos**.

---

## Funcionalidades
- Calcula a integral de uma função fornecida pelo usuário no intervalo `[a, b]`.
- Permite definir o **número de trapézios** e o número de **casas decimais**.
- Calcula o **erro de truncamento** usando a segunda derivada da função.
- Calcula o **erro de arredondamento** baseado no número de trapézios e casas decimais.
- Retorna a **área aproximada**, **erro total** e o **intervalo de confiança** da integral.
